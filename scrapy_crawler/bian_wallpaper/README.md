
爬取彼岸壁纸网站的动漫系列壁纸(1920x1080)
===
**使用方法：**
* 直接运行main.py即可
* 运行结束后会把爬虫结果写入到MongoDB数据库并生成图片文件在result目录下的wallpapers文件夹中
